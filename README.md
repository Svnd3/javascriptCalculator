JavaScript Calculator Application
Overview

This project is a simple calculator application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and keeps a history of all calculations made by the user.

This project was developed as part of a lab assignment to simulate a calculator tool for an operating system.

Features

-Addition

-Subtraction

 -Multiplication

-Division

-Calculation history tracking

-Interactive user interface

 Premium modern UI design

Technologies Used

-HTML5
-CSS3
-JavaScript (Beginner-friendly logic, no eval() used)

How to Run the Project

Download or clone the repository:
git clone git@github.com:Svnd3/javascriptCalculator.git
Open the project folder
Double-click on:
index.html
The calculator will open in your browser

How It Works

Users input expressions like:
5+3
The program:
Detects the operator (+, -, *, /)
Splits the numbers
Performs the calculation using JavaScript logic
Each calculation is stored in an array and displayed in the history section

Project Structure
calculator-app/

│── README.md    # Project documentation

Limitations
-Only supports one operation at a time
Example: 5+3
-Not supported: 5+3*2
-No keyboard input support (button input only)
